----------------------------English----------------------------
code . (open VSCode in this directory)
python3 -m venv venv (Create a Python Virtual environment named venv)
venv/bin/activate (Activate the virtual environment)

pip list (display the current pip version)

pip install --upgrade pip (upgrade pip to the latest version)

(create a file called requirements.txt and write the following two lines: requests, beautifulsoup4, playwright, pandas)
After completing the above step, return to the command line and type: pip install -r requirements.txt (to install the necessary libraries).
